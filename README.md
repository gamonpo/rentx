## How To Use

```
$ git clone https://github.com/gamonpo/rentx.git

$ cd rentx

$ yarn install

$ yarn run android

$ yarn start
```

## Additional notes

In order to run you need to:
1. Download the API
```
$ git clone https://github.com/gamonpo/rentx.git
```
2. Install dependencies
```
$ yarn install
```  
3. Start the server
```
$ yarn start
```
