## How To Use


```bash
# Clone 
$ git clone https://github.com/gamonpo/rentx.git

# Go into
$ cd rentx

# Install dependencies
$ yarn install

# Build your app
$ yarn run android

# Run app
$ yarn start
```

## Additional notes

In order to run you need to:
1. Download the API
```
$ git clone https://github.com/gamonpo/rentx.git
```
2. Install dependencies
```
$ yarn install
```  
3. Start the server
```
$ yarn start
```
